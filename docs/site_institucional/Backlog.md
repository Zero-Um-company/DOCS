## 1. Épicos

| EP-04 | Gestão de formulários e leads | **RF03** O sistema deverá conter um tipo formulário.<br> **RF04** O sistema deverá conter um tipo lead.<br> **RF05** O sistema deverá ser capaz de identificar a qual formulário pertence determinado lead.<br>**RF06** O sistema deverá ser capaz de identificar a qual produto pertence determinado formulário.<br>**RF12** Os usuários deverão ser capazes de se cadastrar no sistema como lead.                                                                                                                                                                                                                                                        |
